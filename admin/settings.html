👤 管理員帳號

在 Firestore 中，admins/{uid} 集合內有文件的使用者即為管理員。 新增管理員請於 Firebase Console 手動建立文件，或透過以下工具指定 UID。

🔧 Firebase 設定說明

1. 前往 Firebase Console 建立專案
2. 啟用 Authentication（Email/Password、Google、Phone）
3. 建立 Firestore Database（建議使用生產模式，再設定安全規則）
4. 將專案設定填入 js/firebase-config.js
5. 部署至 Firebase Hosting：firebase deploy

// Firestore 安全規則範例
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /products/{id} { allow read: if true; allow write: if isAdmin(); }
    match /orders/{id} { allow read, write: if request.auth != null; }
    match /users/{uid} { allow read, write: if request.auth.uid == uid; }
    match /coupons/{id} { allow read: if request.auth != null; allow write: if isAdmin(); }
    function isAdmin() { return exists(/databases/$(db)/documents/admins/$(request.auth.uid)); }
  }
}
